# ⛩️☪️ AstrophileYard 🌟🌊
  <em>For people caught in their mundane life and aspires to be an escapist :))</em> ☄️

A MERN-based web application for space enthusiasts and astrophotographers to explore, upload, and experience cosmic content.

👉 [Visit AstrophileYard](https://astrophileyard.onrender.com)

## 🚀 Overview

AstrophileYard provides:

- Astrophotography archive uploads
- Galactic gallery browsing
- Cosmic audio content streaming
- Smooth navigation and responsive UI

## 🧭 Application Flow
```
User → Signup/Login
↓
Authenticated Session
↓
Upload Archive (MongoDB Storage)
↓
Render Gallery (React Frontend)
↓
Browse / Stream / Explore
```

## ⚙️ Local Setup (Contributors)

### 1️⃣ Fork & Clone

```bash
git clone https://github.com/<your-username>/AstrophileYard-Web-App.git
cd AstrophileYard-Web-App
```

### 2️⃣ Add Upstream
```bash
git remote add upstream https://github.com/Nkovaturient/AstrophileYard-Web-App.git
git remote -v
```

### 3️⃣ Install Dependencies

Ensure:

- Git installed
- Node.js (LTS)

```bash
npm install
```

### 4️⃣ Run Locally
```bash
npm start
```

### 5️⃣ Create Feature Branch
```bash
git checkout -b feature/<feature-name>
```

### 6️⃣ Commit and Push
```bash
git add .
git commit -m "feat: concise description"
git push origin feature/<feature-name>
```

### 7️⃣ Keep Your Branch Updated
1. Fetch and merge the latest changes from the upstream repository:
   ```bash
   git fetch upstream
   git merge upstream/main
   ```

2. Resolve any merge conflicts if necessary.

### 8️⃣ Create a Pull Request & All good
